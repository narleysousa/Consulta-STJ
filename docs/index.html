Como usar

Busca por período: configure os filtros na barra lateral e clique em Consultar processos.

No GitHub Pages, a API pode bloquear a consulta direta no navegador. Para consulta instantânea, use Vercel ou rode ./iniciar.sh localmente.

Busca por número: cole um ou mais números CNJ na aba correspondente.

Detalhes: clique em qualquer linha da tabela para ver o histórico de movimentos.

Critérios

  • Trânsito + Baixa: processos com ambos os movimentos (TPU 848 e 22)
  • Só trânsito / Só baixa / Trânsito ou Baixa: variações do filtro

Processos do Paraná são identificados pelo padrão .8.16. no número CNJ (TJPR).
